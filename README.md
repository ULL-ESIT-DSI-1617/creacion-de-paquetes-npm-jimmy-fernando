# Módulo Shape

Módulo principal Shape del cual heredan las demás clases Square, Triangle y Rectangle.

Si queremos ejecutar el script, en la línea de comandos escribiremos:

```bash
$> ./bin/team-area.js [options] [tipo_de_figura]
```
